<task type="auto">
  <name>Task 2: Create services data access layer</name>
  <files>utils/db/services.js</files>
  <action>
Create utils/db/services.js with the following functions, following the same pattern as courses.js:

1. **getAll()** - Get all services from database, map DB columns to JSON format (order_number -> order, enabled INTEGER -> boolean)
2. **getEnabled()** - Get only enabled services ordered by order_number
3. **getById(id)** - Get service by id (string id from JSON)
4. **update(id, updates)** - Update service fields with validation (name/url cannot be empty)
5. **saveServices(config)** - Save entire config (for compatibility with services-loader.js)
6. **migrateFromJson(jsonPath)** - Migrate from JSON file to SQLite using transaction
7. **validateServices(services)** - Validate services array (same as services-loader.js)

Key mappings:
- JSON 'order' -> DB 'order_number'
- JSON 'id' -> DB 'id' (string)
- DB returns enabled as INTEGER (0/1), convert to boolean
- Use same validation checks as services-loader.js

Use better-sqlite3 for synchronous operations. Use db.transaction() for bulk inserts.
  </action>
  <verify>
<automated>node -e "const svc=require('./utils/db/services'); console.log('getAll:', svc.getAll().length, 'services');"</automated>
  </verify>
  <done>getAll, getEnabled, getById, update, saveServices functions work correctly</done>
