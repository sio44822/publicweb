## Interfaces

From utils/db/schema.js - Statistics table definition:
```javascript
CREATE TABLE IF NOT EXISTS statistics (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  userId TEXT,
  pagePath TEXT NOT NULL,
  timestamp INTEGER NOT NULL,
  created_at INTEGER DEFAULT (strftime('%s', 'now'))
)
```

From utils/db/schema.js - Daily stats table definition:
```javascript
CREATE TABLE IF NOT EXISTS daily_stats (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  date TEXT UNIQUE NOT NULL,
  totalVisitors INTEGER DEFAULT 0,
  totalPageViews INTEGER DEFAULT 0,
  created_at INTEGER DEFAULT (strftime('%s', 'now')),
  updated_at INTEGER DEFAULT (strftime('%s', 'now'))
)
```

## Data Format

coupon-statistics.json structure:
```json
[

]
```

daily-statistics.json structure:
```json
{
  "2026-04-21": { "userCount": 1, "visitCount": 1 }
}
```
